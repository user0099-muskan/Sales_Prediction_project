# Sales Prediction Using Python

This project predicts product sales based on advertising expenditure using Machine Learning techniques in Python.

## Project Overview
The goal of this project is to analyze how different advertising channels affect sales and to build a predictive model using Linear Regression.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Data visualization of advertising spend vs sales
- Sales prediction using Linear Regression
- Model evaluation and accuracy check

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
The dataset contains advertising budgets for:
- TV
- Radio
- Newspaper  

and the corresponding sales values.

## Model Used
- Linear Regression (Supervised Machine Learning)
