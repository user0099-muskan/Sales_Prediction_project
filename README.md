#**Sales Prediction Using Python**
This project predicts sales based on advertising expenditure using Machine Learning techniques in Python.
##**Features**
-Data preprocessing and analysis
-Visualization of advertising data
-Sales prediction using Linear Regression
-Model evaluation and performance analysis
##**Technologies Used**
-Python
-Pandas
-Numpy
-Matplotlib
-Scikit-learn
##**Dataset**
The dataset contains advertising budgets for TV, Radio,and Newspaper along with corresponding sales figures.
